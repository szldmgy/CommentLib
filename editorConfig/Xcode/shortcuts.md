Shortcuts
=========

View
----

`shift + command + Y`: Show or Hide Debug Area

Product
-------

`command + R`: Run

`command + B`: Build

Edit
----

`command + .`: Show completions

`command + [`: 左缩进

`command + ]`: 右缩进

Navigate
--------

`control + command + ←`: Go Back

`control + command + →`: Go Forward

Help
----

`shift + control + command + ?`: Quick Help for Selected Item
