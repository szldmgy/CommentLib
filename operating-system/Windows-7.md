- Chrome 主浏览器。**QQ 电脑管家** > 软件管理 > 软件仓库
- Opera 12 次浏览器。**QQ 电脑管家** > 软件管理 > 软件仓库

- [QQ][2] 沟通
- [gVim][3] 主文本编辑工具

- [QQ 电脑管家][6] 计算机维护

- HDTunePro5.0 硬盘管理。**微云** > Tool
- Reflector7.5.2.1 .NET 反射工具 **微云** > dev > C# > ★ 反编译
