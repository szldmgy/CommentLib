See IP
------

Open terminal (or iTerm):

```bash
$ ifconfig
```

find inet.

List All Files (include hidden file)
-----------------------------------

```bash
$ ls -a
```

Copy File
---------

```bash
$ cp /dir/filename /todir
```

Copy Directory
--------------

```bash
$ cp /dir/* /todir
```

New Line
--------

Sometimes, the bash would be very long, you can break it:

```bash
$ some very very very \
  very very very very \
  very long command
```

`\` means command is not end.
